Wie bekomme ich einen Token?
  1. github.com → Profilbild → Settings → Developer settings
  2. Personal access tokens → Fine-grained tokens → Generate new token
  3. Repository access → "Only select repositories" → padel-fermoselle
  4. Permissions → Repository permissions → Contents → "Read and write"
  5. Token generieren, kopieren, beim ersten Klick auf "Veröffentlichen" hier einfügen
